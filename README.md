# math-painter

[![Powered by DartNode](https://dartnode.com/branding/DN-Open-Source-sm.png)](https://dartnode.com "Powered by DartNode - Free VPS for Open Source")

A minimal, vim-like geometry drawing tool built with TypeScript, Vite, Canvas 2D, and native DOM.

## Development

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:5173
```

## Build

```bash
npm run build
npm run preview
```
